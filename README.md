# NavBar Microservice for AirBnBeats
AirBnBeats is a replica of beatport.com using React, Node/Express, and Mongo/Mongoose

## Usage

1. run npm build
2. run npm seed
3. npm start
4. Go to localhost:3737

## Requirements

- Node
- MongoDB
